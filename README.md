# 🎵 SonicPress

**MP3 壓縮工具 · 讓音樂變小聲，品質不變**

---

## 📦 簡介

SonicPress 是一款輕量級的 MP3 音訊壓縮工具，讓你可以快速壓縮音樂檔案，節省硬碟空間，同時保留良好的音質。

---

## ✨ 功能特色

- 🎯 **拖曳載入** — 直接把音檔拖進去，不用慢慢找
- 🎚️ **位元率控制** — 從 64kbps ~ 320kbps 自由調整
- 📊 **取樣率控制** — 16kHz ~ 44.1kHz 隨你調
- 🔇 **單聲道切換** — 轉成 Mono 直接省 40% 空間
- 📈 **即時對比** — 壓縮前後大小一目瞭然
- 🔊 **試聽功能** — 壓縮完馬上聽效果
- 💾 **一鍵儲存** — 壓縮完直接存檔

---

## 🖥️ 系統需求

- Windows 10 / 11
- 無需安裝 Python
- 支援格式：MP3 / WAV / M4A / FLAC

## 🚀 使用方法

1. 載入音檔（點擊或拖曳）
2. 調整位元率（64k ~ 320k）
3. 調整取樣率（16k ~ 44.1k）
4. 選擇是否轉為單聲道
5. 點擊「壓縮」
6. 試聽或儲存結果

---

## 📊 壓縮效果

| 原始大小 | 壓縮後 | 節省空間 |
|----------|--------|----------|
| 10 MB | 2.5 MB | **75%** |
| 5 MB | 1.2 MB | **76%** |

> 實際效果依音檔內容和設定而異
